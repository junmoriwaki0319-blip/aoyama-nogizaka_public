チーム紹介

青山乃木坂パートナーズ

Philosophy

私たちは、投資家の視点と経営者の感覚を兼ね備えたプロフェッショナルとして、

企業が本来持つべき価値を最大限に引き出すための支援を行います。

提言で終わらせず、実行まで共に走り抜くことを、揺るぎないコミットメントとしています。

Representative

代表

Representative / 代表

森脇 潤

Jun Moriwaki

早稲田大学卒業後、アクセンチュア戦略コンサルティング本部、アーサー・ディ・リトル（ADL）にて経営コンサルティングに従事。SIer・通信・自動車・エンタメ等の業界を中心に、成長戦略・新規事業立案からPEファンド向けBDD・PMIまで幅広くリード。

その後アドバンテッジパートナーズにて上場企業向けエンゲージメント投資に従事し、投資実行からバリューアップ・Exitまでを一気通貫で担当（担当6社）。2025年に青山乃木坂パートナーズを設立。

Members

メンバー

Consultant

アクセンチュア戦略コンサルティング本部に10年間にわたり在籍。その後、東証プライム上場大手企業の経営企画部長（IR・海外担当 兼任）として、成長戦略・中計の策定、事業ポートフォリオの見直し、不採算事業の再建などを推進。

Advisors

アドバイザー

Advisor

メガベンチャーでのIPOを経営企画として実現した後、ヘッジファンド勤務を経験。資本市場・成長戦略の両面でアドバイスを提供予定。

主な専門領域：経営戦略策定 / 新規事業立案 / M&A推進支援 / PEファンド向けBDD・PMI / エンゲージメント投資 / バリューアップ支援 / アクティビスト対策 / IR支援 / 事業ポートフォリオ再編 / 中期経営計画策定
